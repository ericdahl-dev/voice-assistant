# Phase 1 voice provider: Vapi

Vapi is the managed voice platform for Phase 1–2. Chosen over Retell for stronger LLM/voice provider flexibility — swapping between OpenAI Realtime and Grok Voice requires an adapter change, not a platform change.

Vapi is explicitly a Phase 1–2 tool. Phase 3 replaces it with Pipecat or LiveKit Agents under the `VoiceAgentProvider` abstraction.
