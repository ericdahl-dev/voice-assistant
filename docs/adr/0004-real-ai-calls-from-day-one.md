# Skip fake-door MVP; ship real AI calls from day one

Phase 1 ships real AI-powered outbound calls, not a concierge/human-in-the-loop stub. The fake-door approach (human operator + AI summary) was considered for demand validation but rejected — it delays learning about the actual hard problems (voice quality, IVR navigation, escalation, disclosure reactions) and creates a misleading baseline.

Starting with a managed voice platform (Vapi or Retell) keeps Phase 1 infrastructure light while still shipping a real AI call experience.
